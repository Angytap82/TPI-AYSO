# Trabajo Integrador: Arquitectura y Sistemas Operativos (AYSO)

Este repositorio contiene la entrega formal del Trabajo Integrador correspondiente a la **Tecnicatura Universitaria en Programación**. 

## 🎯 Objetivo del Proyecto
Demostrar las competencias adquiridas en la administración de recursos lógicos y la consolidación de servidores mediante un hipervisor de Tipo 2 (**Oracle VirtualBox**), ejecutando un entorno aislado en **Ubuntu Server** para la ejecución de scripts nativos en **Python**.

## 📁 Archivos Incluidos
* `promedio.py`: Código fuente de la aplicación interactiva que calcula el promedio de notas.
* `Trabajo_Integrador_AYSO.pdf`: Informe técnico digital en formato PDF con la fundamentación y anexos de capturas.

## 🚀 Instrucciones de Ejecución en el Servidor
1. Iniciar la máquina virtual `Servidor-AYSO` en VirtualBox.
2. Iniciar sesión con las credenciales configuradas (`alumno`).
3. Ejecutar el script interactivo mediante el comando:
   ```bash
   python3 promedio.py
   ```
